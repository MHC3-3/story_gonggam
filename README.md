## Getting Started

### 0. Clone 및 패키지 설치

```
git clone https://github.com/MHC3-3/story_gonggam.git
cd story-gonggam
yarn
```

### 1. 환경변수 설정

- .env.local.sample 파일 참조

```jsx
NEXT_PUBLIC_MAP_KEY=
NEXT_PUBLIC_KAKAO_JAVASCRIPT_KEY=
NEXT_PUBLIC_COUNTUP=
```

### 2. start

```jsx
yarn dev
```
